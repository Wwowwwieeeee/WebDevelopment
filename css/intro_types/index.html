Introduction to CSS

CSS stands for Cascading Style Sheet. It is used to style the .html files. There are three different types of CSS. They are:
  • Inline CSS
  • Internal CSS
  • External CSS
Inline CSS
Inline CSS is the type of CSS which is used in .html files. It is defined in .html files by using style attributes.
            For example: In html files:
                
                  <h1 style="background colour="red" colour="white""> Introduction </h1>
Internal CSS
Internal CSS is alsso declare in .html files above the body tag. It is used when the code is short. Otherwise it will make the actual html code long which makes it hard to understand and maintain.
            
            
  For example:
            <head>
              <title>Introduction</title>
            </head>
            <style>
               div{
                background-colour="Blue";
                colour="white";
               } 
            </style>
External CSS
 External CSS is declared in seperate file using extension ".css" and add all your styles in that file. Then you attach these .css file in html file by using: 
            
             "<link rel="stylesheet" href="">"
**Note** Inline CSS has background color red and color white whereas Internal CSS has colour blue. All aqua colour and also this note is fron external CSS.
